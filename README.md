# AIgnite Website

This is the official website for AIgnite, the AI club of IILM University, Gurugram. The site is built using HTML, CSS, and JavaScript, and showcases club information, events, team members, and contact details.

## Project Structure

- `index.html` — Main webpage
- `css/style.css` — Stylesheet for the site
- `js/script.js` — JavaScript for interactivity
- `images/` — Club and site images
- `README.md` — Project documentation
- `LICENSE` — License information

## Features

- Responsive design for desktop and mobile
- Modern UI with custom styles
- Team and event sections
- Contact information

## Usage

Simply open `index.html` in your web browser. No backend or server is required.
